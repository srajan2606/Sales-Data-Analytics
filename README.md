# Sales Data Analytics Report

This repository contains an interactive **Power BI** dashboard designed to analyze sales performance, profitability trends, and product distributions.

## 📊 Dashboard Overview

The Power BI report consists of five specialized analytical tabs:

### 1. Sales Customers Analytics
Provides a high-level summary of customer purchasing behaviors and macro sales patterns.
* **Key Performance Indicators (KPIs):** Sum of Net Sales, Profit vs. Net Sales matrix, and Sales Trend by Period.

### 2. Top/Bottom 5 Products Analysis
Identifies the highest and lowest performing inventory items based on financial performance to aid in stock optimization and marketing focus.

### 3. Comparison of Sales/Profit/Quantity
Features side-by-side comparative visualizations evaluating the relationship between revenue generated, net profit margins, and total units sold.

### 4. Edit Interaction
Demonstrates advanced dashboard filtering behavior, allowing users to cross-filter or isolate specific data points seamlessly across visuals.

### 5. Table Visual
A comprehensive tabular view containing detailed records for granular data exploration, auditing, and export functionality.

## 🛠️ Tools Used
* **Power BI:** Data modeling, DAX queries, and interactive report development.

## 🚀 How to View the Report
1. Download the sales analytics `.pbix` file from this repository.
2. Open the file using [Power BI Desktop](https://powerbi.microsoft.com/desktop/) to explore the interactive visual elements.
